# 🌐 Personal Portfolio Website

A fully responsive personal portfolio website built from scratch using **HTML5, CSS3, and vanilla JavaScript** — no frameworks, no templates.

Live at: [sohail-logics.netlify.app](https://sohail-logics.netlify.app)

---

## ✨ Features

- 🌗 Light / Dark mode toggle with localStorage memory
- 📱 Fully responsive — mobile, tablet, and desktop
- 🎨 Smooth scroll animations and reveal effects
- 🧩 Bento-style About section (bio, age, location, hobbies, socials, languages)
- 💼 Projects showcase with GitHub repository links
- 🎓 Education timeline with current degree highlighted
- 🏅 Certifications section with filter system — opens in a dedicated page
- 📬 Contact form connected directly to email via mailto
- ⚡ Auto-deployed via Netlify — any GitHub push goes live in under 2 minutes

---

## 🛠️ Built With

- HTML5
- CSS3 (CSS Variables, Grid, Flexbox)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Fraunces + Plus Jakarta Sans)

## 🚀 Getting Started

No installation needed. Just open `index.html` in any browser.

Or view it live at: [sohail-logics.netlify.app](https://sohail-logics.netlify.app)

---

## ✏️ How to Customize

| What | Where |
|------|-------|
| Your photo | `images/` folder → update `src` in hero section |
| Your name / bio | `index.html` → About section |
| Projects | `index.html` → Projects section |
| Certifications | `certifications.html` → update each card |
| Resume PDF | `assets/` folder → update resume button link |
| Social links | Search `sohail-logics` and replace with your username |
| Color theme | CSS variables at top of `<style>` tag |
